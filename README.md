
# 💀 F-SOCUETY 404 - Advanced Cyber Suite

F-SOCUETY 404 একটি উচ্চক্ষমতাসম্পন্ন এবং আধুনিক পাইথন ভিত্তিক ওয়েব হোস্টিং টুল। এটি ডেভেলপার এবং এথিক্যাল হ্যাকারদের জন্য তৈরি করা হয়েছে, যা খুব দ্রুত এবং সহজে যেকোনো ওয়েব কনটেন্ট ইন্টারনেটে লাইভ করতে সাহায্য করে।



## ✨ মূল বৈশিষ্ট্যসমূহ (Features)
* **লোকাল ফাইল হোস্টিং:** লোকাল স্টোরেজের HTML ফাইল মুহূর্তের মধ্যে লাইভ করুন।
* **ওয়েব ইউআরএল স্ক্র্যাপিং:** যেকোনো ওয়েবসাইটের লিংক থেকে কনটেন্ট কপি করে হোস্ট করুন।
* **নিরাপদ হোস্টিং:** সাইটে পাসওয়ার্ড প্রটেকশন যোগ করার সুবিধা।
* **ভিজিটর ট্র্যাকিং:** স্মার্ট ভিজিটর কাউন্টার।
* **অটো-ব্রাউজার লঞ্চার:** হোস্টিং সাকসেসফুল হওয়ার সাথে সাথে ব্রাউজারে লিংক ওপেন হবে।
* **টেলিগ্রাম নোটিফিকেশন:** হোস্টিং করা সাইটের লিংক এবং ফাইল স্বয়ংক্রিয়ভাবে আপনার টেলিগ্রাম বট-এ পাঠিয়ে দেবে।

## 🛠 ইনস্টলেশন পদ্ধতি (Installation)
আপনার ডিভাইসে (Termux বা Linux) টুলটি সঠিকভাবে সেটআপ করতে নিচের কমান্ডগুলো সিরিয়ালে রান করুন:

১. **প্রজেক্টটি ডাউনলোড (Clone) করুন:**
```bash
git clone [https://github.com/DEVIL-DOOR/F-SOCUETY-HOST-404](https://github.com/DEVIL-DOOR/F-SOCUETY-HOST-404)

২. প্রজেক্ট ফোল্ডারে প্রবেশ করুন:
cd F-SOCUETY-HOST-404

৩. প্রয়োজনীয় লাইব্রেরি ইনস্টল করুন:
pip install requests

৪. টুলটি চালু করুন:
python socuety-hoster.py

💻 ব্যবহারের নিয়ম (Usage Guide)
টুলটি চালু হওয়ার পর আপনি মেনু থেকে আপনার পছন্দমতো অপশন (১ থেকে ৪) সিলেক্ট করুন। হোস্টিং সফল হওয়ার সাথে সাথে আপনার ব্রাউজারে সাইটটি ওপেন হবে এবং আপনার টেলিগ্রাম বটে রিপোর্ট চলে যাবে।
